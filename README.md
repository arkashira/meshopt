# Meshopt

A unified dashboard for visualizing and managing service meshes.

## Features

* Displays a list of all connected service meshes
* Provides real-time metrics and monitoring data for each service mesh
* Allows for filtering and sorting of service meshes by name, type, and status

## Usage

1. Create a `Meshopt` instance
2. Add service meshes using `add_service_mesh`
3. Get the list of service meshes using `get_service_meshes`
4. Filter service meshes using `filter_service_meshes`
5. Sort service meshes using `sort_service_meshes`
6. Get metrics for a service mesh using `get_metrics`
